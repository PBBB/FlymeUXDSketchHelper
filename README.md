# Fletch

## 功能列表
1. 自动更新页码、目录和日期
2. 添加更新记录
3. 显示或隐藏界面框或逻辑说明，并重排字母
4. 添加新的文档画板
5. 导出并压缩 PDF（多线程）
6. 支持自定义的工具栏

## 脑洞功能，不一定做
1. 从一个文档格子中复制到另一个文档格子时，保持相同的相对文档格子的位置
2. 页面与画板的互相转换（单独的画板容易在手机预览）
3. 自动生成交互连线
4. 添加 UIKit 中的 symbol
